# DevelopersHub-DS-Analytics-Internship-Phase-2
Data Science Internship Tasks (DevelopersHub) Phase 2
DevelopersHub Data Science & Analytics Internship

Task 1: Term Deposit Subscription Prediction
Dataset: bank.csv
Description: Classification models (Logistic Regression, Random Forest) used to predict whether a customer subscribes to a term deposit. Includes data preprocessing, encoding, and evaluation using Confusion Matrix, F1-score, and ROC Curve.
Accuracy: ~85–90% (depending on model)
Insight: Features like duration, previous campaign outcome, and contact type strongly influence subscription decisions.

Task 2: Customer Segmentation
Dataset: mall/customers dataset (or tips dataset used)
Description: K-Means Clustering applied to segment customers based on spending behavior. Visualization done using scatter plots.
Insight: Customers can be grouped into low, medium, and high spenders, enabling targeted marketing strategies like discounts for low spenders and premium offers for high spenders.

Task 5: Business Dashboard (Streamlit)
Dataset: superstore.csv
Description: Interactive Streamlit dashboard to analyze sales and profit across regions with filters and visualizations like bar charts for top customers.
Insight: Certain regions and customers contribute significantly more to revenue, helping businesses focus on high-value segments.
